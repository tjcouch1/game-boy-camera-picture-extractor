# Frames

This folder contains frames for the Game Boy Camera website.

There are two possible subfolders. Each subfolder should have subfolders containing frame images.

- `individual/**/*.{png,etc}` - individual frame images. The 128x112 hole where the Game Boy Camera image should go should be completely white or transparent in order to be detected properly
- `sheets/**/*.{png,etc}` - sprite sheets containing multiple frames. The top left corner should have the background color. The 128x112 hole where the Game Boy Camera image should go should be the background color
